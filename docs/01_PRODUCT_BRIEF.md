# 01_PRODUCT_BRIEF

## Product Identity

**Nightseed Bastion** is a premium-feeling vertical mobile strategy action game. The player commands a lone ruler defending a cursed fortress from waves of nightborn enemies.

## Target Player

- Likes tactical decisions but dislikes heavy RTS micromanagement.
- Wants short but meaningful mobile sessions.
- Enjoys dark fantasy pixel art.
- Likes progression, unlocks, relic choices, and repeated runs.

## Session Length

| Mode | Target Length |
|---|---:|
| One Night | 90-150 seconds |
| One Full Run | 12-18 minutes |
| Failed Run | 4-12 minutes |

## Primary Differentiators

1. **Vertical one-thumb strategy:** mobile-first layout instead of PC-first RTS UI.
2. **Dusk Omen:** enemies and lanes are previewed before combat.
3. **Dusk Bargain:** take risky cursed offers before night begins.
4. **Hero + fortress hybrid:** player acts directly while buildings handle defense logic.
5. **Run-based bastion evolution:** each run creates a temporary fortress, while meta progression unlocks future choices.

## Monetization Baseline

Initial recommendation:

- Paid app or free demo + one-time full unlock.
- No forced ads in the first release.
- No gacha.
- No pay-to-win upgrades.

Rationale: the game depends on trust, clarity, and tactical purity. Ads and aggressive economy design would weaken the intended product identity.
