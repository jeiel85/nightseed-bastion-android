## Store Description Draft - Korean

밤의 씨앗이 깨어나면, 마지막 성채만이 남습니다.

Nightseed Bastion은 세로 화면에 맞춘 다크 판타지 요새 생존 전략 게임입니다. 낮에는 성채를 확장하고, 황혼에는 적의 징조를 읽고, 밤에는 직접 영웅을 조작해 몰려오는 군세를 막아내세요.

특징:

- 세로 화면 모바일에 최적화된 요새 방어 전략
- 낮/황혼/밤/새벽으로 이어지는 선명한 전투 루프
- 영웅 조작과 자동 방어 건물의 조합
- 위험한 보상을 선택하는 Dusk Bargain 시스템
- 반복 플레이로 확장되는 영웅, 건물, 유물, 도감

## Store Description Draft - English

When the Nightseed awakens, only the last bastion remains.

Nightseed Bastion is a vertical dark-fantasy fortress survival strategy game. Build your stronghold by day, read enemy omens at dusk, and command your hero directly at night while towers, traps, and walls hold the line.

Features:

- Mobile-first vertical fortress defense
- Day, dusk, night, and dawn tactical loop
- Hybrid hero action and automatic defenses
- Risk-reward Dusk Bargain system
- Unlockable heroes, buildings, relics, and codex entries
