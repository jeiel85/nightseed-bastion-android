# SOURCE_NOTES.md

This pack adapts the user's existing agent-governance style into a Nightseed Bastion-specific repository workflow.

Applied governance ideas:

- Automation-first development with explicit stop conditions
- Korean-first communication and commit summaries
- Strict changelog and history maintenance
- Mobile release asset verification: APK, AAB, release notes, version sync
- Security guardrails for signing keys, SDKs, external services, and store credentials
- Explicit distinction between verified results and unverified assumptions

This package intentionally excludes Nightseed Survivor design content. Nightseed Survivor is treated as an already released separate project and is not part of this repository's game design scope.
